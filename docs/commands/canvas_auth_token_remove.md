## canvas auth token remove

Remove API token from an instance

### Synopsis

Remove the API token from a Canvas instance configuration.

This removes token-based authentication. If the instance also has OAuth
credentials, you can still use 'canvas auth login' to authenticate.

Examples:

```bash
canvas auth token remove myschool
```

```
canvas auth token remove <instance-name> [flags]
```
